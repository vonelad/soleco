# @soleco/lucide

> Lucide icons as Solid components

## Install

```bash
# npm
npm install --save @soleco/lucide

# yarn
yarn add @soleco/lucide

# pnpm
pnpm add @soleco/lucide
```

## Usage

```jsx
import Bookmark from "@soleco/lucide/bookmark";

const App = () => (
  <div>
    <Bookmark />
  </div>
);
```
